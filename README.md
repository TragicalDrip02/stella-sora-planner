# Stella Sora - Ascension Planner

A web-based tool for planning character ascension, leveling, and skill upgrades for the game Stella Sora.

## File Structure


## Features

- Character management (add, edit, complete, hide)
- Material requirement calculations
- Inventory tracking
- Dark/light theme
- Export/import functionality
- Responsive design

## Usage

1. Open `index.html` in a web browser
2. Click "Add Character" to start planning
3. Use "Manage Inventory" to track owned materials
4. Toggle between "Total Needed" and "Remaining Needed" views

## Browser Compatibility

Works in modern browsers with ES6+ support.
